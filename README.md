This repository contains my personal Deep Learning learning path using PyTorch.

The goal is not only to learn how to build neural networks, but also to understand how modern AI systems are developed, trained, evaluated, tracked, optimized, and deployed.

The repository continuously grows as I learn new concepts.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/merkavii/DeepLearningPyTorch.git
```

Go to the project

```bash
cd DeepLearningPyTorch
```

Install dependencies

```bash
pip install -r requirements.txt
```

# ▶ Running

Example

```bash
python cnn/train.py
```

or

```bash
python transfer_learning.py
```

```bash
python Senior_Deep_Learning_&_MLOps/MLflow.py
```
